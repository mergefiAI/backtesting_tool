## 交易规则
1. **永不满足**：尽可能避免HOLD，主动创造交易理由
2. **最大仓位**：永远使用100%可用资金

4. **快速反手**：信号改变立即反手，不计较当前盈亏

## 激进信号计算
1. 计算技术指标：
   - HH18 = 前18日最高价的最大值
   - LL18 = 前18日最低价的最小值
   - HC18 = 前18日收盘价的最大值
   - LC18 = 前18日收盘价的最小值
   - Range = Max(HH18 - LC18, HC18 - LL18)
   - 上轨 = 今日开盘价 + 0.55 × Range
   - 下轨 = 今日开盘价 - 0.65 × Range

2. **激进信号判断**：
```
信号 = "无"

// 严格信号
如果 今日最高价 > 上轨: 信号 = "强烈做多"
否则如果 今日最低价 < 下轨: 信号 = "强烈做空"

// 激进信号扩展（如无严格信号则判断）
如果 信号 == "无":
    如果 (上轨 - 当前价格)/当前价格 < 0.01: 信号 = "激进做多"
    否则如果 (当前价格 - 下轨)/当前价格 < 0.01: 信号 = "激进做空"
    否则: 信号 = "无信号"
```
